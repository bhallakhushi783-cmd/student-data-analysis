# Sales Data Analysis using Pandas

## Overview
This project analyzes sales data using Python and Pandas and matplotlib.

## Features
- Calculate total sales (Price × Quantity)
- City-wise total sales 
- Product-wise total sales
- Identify top-selling product in each city
- Data visualization (bar charts)

## Tools Used
- Python
- Pandas
- Matplotlib

## Dataset
Sample sales dataset with columns:
OrderID, Product, Category, Price, Quantity, City, Date

## Key Insights
- Delhi generated the highest total sales among all cities.
- Laptop is the top-performing product based on total sales.

## How to Run
1. Place `sales_data.csv` in the same folder
2. Run the script:
3. python dashboard.py
