# Sales Data Analysis using Pandas

## Overview
This project analyzes sales data using Python and Pandas.

## Features
- Calculate total sales (Price × Quantity)
- City-wise total sales analysis
- Identify top-selling product in each city

## Tools Used
- Python
- Pandas

## Dataset
Sample sales dataset with columns:
OrderID, Product, Category, Price, Quantity, City, Date

## Key Insights
- Delhi has highest sales
- Laptop is top-performing product in most cities

## How to Run
1. Place `sales_data.csv` in the same folder
2. Run the script:
3. python main.py
